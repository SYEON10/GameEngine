### Goal 1

에디터 상에서 씬에 3D 오브젝트를 띄우고 상용 엔진과 유사한 기본 조작을 수행할 수 있게 한다.
이 과정에서 기본적인 게임 루프, 렌더링, 오브젝트 관리에 대한 기반을 수립한다.

#### 사전 세팅
- [ ] .editorconfig 설정
- [ ] 전역 헤더 구성

#### 코어 구현
- [ ] 언리얼 방식 기반의 UObject 구현(RTTI 포함)
- [ ] 개발에 사용할 기본 자료구조 구현
- [ ] 메인 루프 구현
- [ ] 3차원 공간에 오브젝트 배치
    - [ ] MVP 변환 구현
- [ ] 씬 생성, 저장, 로드 (.umap으로 구현)

#### 에디터 구현
- [ ] 씬 구현
- [ ] 씬에서 오브젝트 클릭 시
    - [ ] 기즈모 구현
    - [ ] Transform 정보를 띄울 Inspector 창
    - [ ] Vertex Shader로 선택한 오브젝트를 강조
- [ ] 카메라
    - [ ] WASD와 마우스로 카메라를 이동
- [ ] 언리얼 Stat 창을 참고하여 유의미한 지표 출력
